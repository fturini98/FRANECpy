Modules
=======

This library is divided into three modules, one that manages the retrieving of the file path of data files fromm your PC (**browse_and_choose_file_paths**), one that manages the creation,saving and loading of the data in a tree format (**build_tree**), one that provides the interface to the user (**user_interaction**).
